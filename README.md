
# Stone-Paper-Scissors Game

## Description
This project implements the classic Stone-Paper-Scissors game using HTML, CSS, and JavaScript. Players choose between stone, paper, or scissors, and the outcome is determined based on predefined rules.

## Deployment
The game is deployed at: [Stone-Paper-Scissors Game](https://rishabh022.github.io/Stone-Paper-Scissors/)


## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Single-player mode against a computer opponent.
